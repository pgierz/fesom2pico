Detailed Description

Makes PICO input from FESOM output!

This script sets up a SLURM job which is able to generate a PISM capable input for the PICO ocean component from a FESOM output

Elements of the script can be used seperately, you just need to source the file and call the functions that you might need.

Function Documentation

extract_fesom_data ( )

Gets relevant FESOM data from a experiment folder.

Gets the FESOM output and combines it to a common file. We assume that there are variables set:

  1. FESOM_OUTPUT_DIR (required)
  2. TEMP_VARNAME (required if ESM_STYLE is not yes)
  3. SALT_VARNAME (required if ESM_STYLE is not yes)
  4. ESM_STYLE (default "yes")
fesom_scalar_to_LatLon ( )

Runs the python program fesom_scalar_to_LatLon.

Arguments:

  • $4: FESOM_OUTPUT (defaults to a modifed version of ${FESOM_FILE}
